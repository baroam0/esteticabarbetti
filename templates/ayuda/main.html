Descripción General

Información General

Esta aplicación web permite gestionar de forma centralizada los tratamientos, productos, turnos y reportes. Su objetivo es ofrecer una plataforma clara y eficiente para la administración diaria de información operativa.

Tipos de Usuario
Administrador

Posee acceso total a todos los módulos del sistema. Puede crear, editar, eliminar y visualizar cualquier registro, además de gestionar configuraciones generales.

Usuario Común

Cuenta con acceso restringido. Solo puede visualizar y operar en los módulos habilitados para su rol, sin permisos administrativos.

Estructura de Navegación

La aplicación utiliza un menú lateral compuesto por elementos llamados Módulos. Cada módulo representa una sección funcional del sistema.

  • Tratamientos: muestra un listado de tratamientos disponibles.
  • Productos: permite visualizar y gestionar el inventario.
  • Turnos: presenta un calendario interactivo para administrar citas.
  • Reportes: ofrece acceso a informes generados a partir de los datos del sistema.

Excepto el módulo de Turnos, cada sección inicia mostrando un listado desde el cual el usuario puede crear nuevos registros, editar información o consultar detalles, según sus permisos.
